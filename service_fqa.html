帳號問題

Q：請問我忘記黑馬會員密碼要如何查詢?

A：查詢會員密碼的步驟如下：

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 點選【忘記密碼】。
  3. 輸入您要查詢的帳號，並點選【查詢】。
  4. 請至您的帳號信箱收信，即可獲得一組新的密碼。

Q：請問已經忘記會員帳號，我要在那裡查詢?

A：若您目前已經忘記帳號，請於【黑馬數位】官網客服中心→【線上回報】提供當時註冊所登記的1.會員姓名2.身分證字號3.手機號碼4.伺服器5.角色名稱，客服專員接收到訊息後，會回覆您並協助查詢帳號。

Q：如何建立一個新的黑馬會員帳號?

A：加入會員的步驟如下：

  1. 請至【黑馬數位】官網，於會員中心點選【加入會員】。
  2. 請務必正確填寫【E-mail帳號】和【會員密碼】資料後，並確實瀏覽會員管理規章，並勾選同意後，點選【確認送出】。
  3. 成為黑馬平台會員。

Q：我有黑馬註冊帳號，要如何成為認證會員?

A：成為認證會員的步驟如下

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 輸入您的E-mail帳號、密碼，並點選【登入】。
  3. 點選【會員中心】填寫會員認證資料後，並點選【確認送出】。
  4. 系統依照會員所填寫的手機號碼進行認證後，點選【確認送出】。
  5. 經過系統驗證後，玩家會收到一則認證碼。
  6. 回到【黑馬數位】官網輸入簡訊認證碼後，點選【確認送出】。
  7. 成為黑馬認證會員。

Q：請問我要在那裡修改我的會員資料?

A：修改會員資料的步驟如下：

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 輸入您的E-mail帳號、密碼，並點選【登入】。
  3. 點選【會員中心】，即可在會員資料進行修改。
    (※會員資料一旦註冊後，只有備用E-mail可以修改)

Q：請問我要在那裡修改我的會員密碼?

A：修改會員密碼的步驟如下：

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 輸入您的E-mail帳號、密碼，並點選【登入】。
  3. 點選【會員中心】。
  4. 點選【修改密碼】。
  5. 輸入您的舊密碼與新密碼，並點選【確認送出】。

Q：會員資料當時註冊時是亂填寫的，還能再修改嗎?

A：會員資料一旦註冊後，只有備用E-mail可以修改，則其它項目(如：真實姓名、帳號 E-MAIL、手機號碼、出生年月日、身分證字號)是無法進行修改的，為了確保會員的權益，建議您在加入會員時請正確填寫會員資料。

Q：請問有黑馬帳號之後，還要註冊遊戲帳號嗎?

A：註冊成為黑馬會員，即可登入黑馬平台所提供的遊戲。

Q：為何使用忘記密碼會收不到新密碼?

A：使用【忘記密碼】功能後，可前往當時註冊郵件收信確認，若您在【收件匣】找不到黑馬數位所發送密碼變更通知的信件，建議您可嘗試至垃圾郵件瀏灠，有可能被郵件系統判定為垃圾郵件，才會歸納在垃圾郵件裡。

寄送的郵件跑到垃圾郵件有以下幾個可能的原因：

  1. 短時間寄送異常過量的信件。
  2. 您的連絡人信箱帳號"有效度"，如：錯誤的信箱帳號的退信比例過高，因為寄給不存在的使用者會被視為攻擊。
  3. 您的連絡人對特定IP、寄件者或網域寄出的信，按 “這是垃圾信”或輸入想擋信的電子信箱(或網域)設定在擋信名單內。
  4. 您的信件本文含有垃圾字串、字典式攻擊、或危險性高的URL。

Q：請問一組手機號碼，可以認證幾組黑馬會員帳號呢?

A：一組手機號碼只能認證一組黑馬會員帳號。

儲值問題

Q：請問什麼是電子發票?

A：配合政府推行『二代電子發票整合政策』並響應環保，依據 【電子發票實施作業要點修正規定】(台財資字第1020002795號)，本站提供消費者可選擇開立「二聯式電子發票」，發票依據會員選擇之載具(共通性載具/黑馬會員載具)，存放於財政部電子發票平台或本站，中獎時由平台或本站通知得獎人索取紙本兌獎。

相關資料請參考 財政部電子發票整合服務平台

  • 依據法令規定，已捐贈之發票無法索取正本及兌獎，捐贈發票電子檔的發票號碼末二碼以"**"代替。
  • 個人戶發票開立後就不能轉為公司戶發票。
  • 開立二聯式紙本發票與三聯式紙本發票，皆直接以紙本寄送。
  • 發票電子檔為副本，依財政部令 發票副本僅供參考，不可直接兌獎。
  • 若您使用手機、市話、ADSL、點數及Apple信用卡機制等付費方式，則發票都由代收廠商處理，請個別聯繫您的代收廠商詢問與索取發票。

Q：請問二聯式電子發票中的『共通性載具』是什麼？

A：配合政府推行『二代電子發票整合政策』，用戶可於【 財政部電子發票整合服務平台】申請『共通性載具』(本站提供使用『自然人憑證(載具)』『手機條碼(載具)』)，於本站購買加值服務時，在共通性載具欄位上輸入條碼號碼(包含英數、符號等)，經驗證通過，本站於訂單付款完成後，將該筆發票以電子檔形式存放於【 財政部電子發票整合服務平台】，由該平台保管與中獎通知，用戶可前往平台查詢發票紀錄，得獎用戶可於平台上設定自動匯款或是到便利商店列印出中獎發票。

Q：我選擇電子發票的話，如何知道我有中獎?

A：選擇『共通性載具』開立之電子發票，由【 財政部電子發票整合服務平台】保管與通知中獎，得獎用戶請上平台查詢。

選擇『黑馬會員載具』開立之電子發票，電子檔為副本，非中獎發票不提供索取列印紙本。凡中獎發票【黑馬數位】會由系統自動兌獎，並EMAIL通知中獎資訊且以紙本發票寄送。

※ 儲值時所要求登記的資料是為了『發票開立』時使用。

Q：如何在【黑馬數位】官網進行儲值?

A：官網儲值的步驟如下：

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 輸入您的E-mail帳號、密碼，並點選【登入】。
  3. 點選【儲值購點】。
  4. 選擇您要的付款方式即可。

Q：請問儲值購點的方式有那幾種?

A：儲值購點方式如下：

  1. MyCard：點卡儲值及轉點和電信小額付費。
  2. 歐付寶：信用卡、WebATM、ATM轉帳、海外儲值。

Q：請問我已經有了點數卡了要怎麼去儲值?

A：點數卡儲值的步驟如下：

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 輸入您的E-mail帳號、密碼，並點選【登入】。
  3. 點選【儲值購點】。
  4. 點選【點卡儲值及轉點】。
  5. 選擇【遊戲名稱】。
  6. 選擇【伺服器名稱】。
  7. 選擇【角色名稱】。
  8. 選擇【MyCard前往儲值點數卡】，輸入上線密碼即可。

Q：信用卡如何儲值?

A：信用卡儲值步驟如下：

  1. 進入【黑馬數位】官網後，請在登入會員的欄位輸入您的黑馬會員帳號及密碼，以完成登入會員。 若您還沒有黑馬帳號，請點這裡【 申請加入黑馬平台會員】。
  2. 登入會員(手機已驗證)後，請在【儲值購點】的頁面選擇【信用卡】儲值方式。
  3. 請選擇您要儲值的遊戲、伺服器、角色名稱。
  4. 請選擇您要儲值的面額。
  5. 點選【前往儲值】後，輸入本次於付費流程中留下手機號碼所收取到的簡訊認證碼並傳送，請注意大小寫。
  6. 經過系統認證後，輸入您的信用卡相關資訊，請確實填妥相關信用卡資料。
  7. 完成儲值。

Q：電信小額付費如何儲值?

A：電信小額付費儲值步驟如下：

  1. 進入【黑馬數位】官網後，請在登入會員的欄位輸入您的黑馬會員帳號及密碼，以完成登入會員。 若您還沒有黑馬帳號，請點這裡【 申請加入黑馬平台會員】。
  2. 登入會員(手機已驗證)後，請在【儲值購點】的頁面選擇【信用卡】儲值方式。
  3. 請選擇您要儲值的遊戲、伺服器、角色名稱。
  4. 請選擇您要儲值的面額。
  5. 請選擇您所使用的電信業者。
  6. 點選【前往儲值】後，請確認您要付費的品項與交易資料，以及門號和同意使用者條約相關內容。
  7. 經過系統認證後，請在3-10分鐘內回覆認證碼，若非在時間內認證完成，請您重新認證。
  8. 完成儲值。

Q：ATM轉帳如何儲值?

A：ATM轉帳儲值步驟如下：

  1. 進入【黑馬數位】官網後，請在登入會員的欄位輸入您的黑馬會員帳號及密碼，以完成登入會員。 若您還沒有黑馬帳號，請點這裡【 申請加入黑馬平台會員】。

  4. 請選擇您要儲值方式和面額，並按下【確認送出】。
  5. 經過系統認證無誤後，您將會收到一組指定轉帳的帳號，並依照指定帳戶前往您家中附近設有ATM進行轉帳。
  6. ATM轉帳成功。

Q：WebATM如何儲值?

A：WebATM儲值步驟如下：

  1. 進入【黑馬數位】官網後，請在登入會員的欄位輸入您的黑馬會員帳號及密碼，以完成登入會員。 若您還沒有黑馬帳號，請點這裡【 申請加入黑馬平台會員】。
  2. 登入會員(手機已驗證)後，請在【儲值購點】的頁面選擇【信用卡】儲值方式。
  3. 請選擇您要儲值的遊戲、伺服器、角色名稱。
  4. 請選擇您要儲值的面額。
  5. 請選擇您要儲值的銀行，並使用讀卡機插入您要儲值的晶片金融卡。
  6. 點選【前往儲值】後，請確認您的儲值相關資訊和要轉出的內容，並按下【確認付費】。
  7. 完成儲值。

Q：儲值成功後點數可以到那裡查詢?

A：建議可至遊戲中查詢您儲值後的點數。

Q：儲值成功後點數沒入帳要怎麼辦?

A：若您已經儲值成功，但無任何點數入帳，這方面請您至【黑馬數位】官網→客服中心→線上回報，與客服專員聯繫，我們將為您做查詢的服務。

Q：儲值點數後是否有使用的期限?

A：儲值成功的點數是可以永久使用的。

Q：我買的點數卡在儲值前不小心將序號遺失了，可以幫我查詢嗎?

A：如果您是購買實體點卡的玩家，未儲值前，請務必保留實體點卡。若點卡遺失客服專員將無法提供查詢服務。如果您是線上購卡的玩家，可以至點數卡官網儲值服務中的帳號查詢，自行查詢您的購卡記錄。

Q：為什麼我的點數卡無法儲值成功?

A：請您確認輸入的點數卡序號無誤，並確保您是使用半形輸入，仔細檢查後再輸入試看看，如果還是無法儲值成功請至【黑馬數位】官網客服中心→線上回報留下您的問題，客服專員收到您的訊息後會為您協助處理。

Q：請問ATM儲值轉帳多久可以入帳？

A：轉帳交易成功後，系統將會在十分鐘之內，將您所匯的金額自動轉為等值的遊戲專用點數。

  3. WebATM：150點、350點、500點、1000點、2000點、5000點、10000點、20000點、30000點。
  4. ATM轉帳：150點、350點、500點、1000點、2000點、5000點、10000點、20000點、30000點。
  5. 海外儲值：150點、350點、500點、1000點、2000點、5000點、10000點。

Q：如何索取發票?

【黑馬數位】邀您一起做公益，每一張發票都代表一份愛心，當您購買加值服務時，玩家們的愛心發票都將捐給公益團體。若您需索取發票時，請登入會員後至【線上回報】系統，告知購買人資料，【黑馬數位】將會把紙本發票郵寄給您。

發票索取注意事項如下：

  1. 請提供購買人資料「開立二聯或三聯式紙本發票、日期、金額、買受人、統一編號、地址」。
  2. 發票寄送將於訂單成立後次月十五日前統一寄出紙本發票。
  3. 未索取發票者本公司將視為同意捐贈【愛心發票】，而捐贈單位為【創世基金會】。
  4. 為了確保您的權利，請填寫正確的購買人資料，若因資料填寫不完整或錯誤導致發票無法寄送或寄錯地址，本公司視同為捐贈【愛心發票】將不另行補發。

活動問題

Q：我在得獎名單中有看到我的帳號角色資訊，請問要如何領獎?

A：當玩家參與活動獲得獎勵時，可透過黑馬數位官網→點選【 新聞專區】→點選【活動】公告→點選您參與的活動名稱和項目→請務必詳細瞭解活動相關內容和注意事項，並依照活動的指示來領獎，以及不定時留意官網相關公告訊息，以確保您自身領獎的權益。

若您對於活動領獎仍有問題，建議您也可透過【 線上回報】系統與我們聯繫說明，客服人員接獲您的回報後會盡快回覆您。

安全問題

Q：會員帳號被盜該怎麼辦?

A：如果發現帳號被盜請您先行至黑馬數位官網【客服中心】→【盜用處理】查閱相關盜用處理流程和內容，並提供相關的資料及證明，若查明屬實客服中心會立即為您處理後續相關程序且視情況將該帳號和相關涉案帳號暫時停權。

Q：我的帳號好像被盜了，請問我要怎麼索回密碼?

A：親愛的玩家請您不用擔心，若您的帳號密碼遭修改，建議您可透過【黑馬數位】官網→點選左側【登入會員】使用【忘記密碼】功能，系統將寄送到您註冊時填寫的信箱並隨機產生一組新的密碼，而註冊信箱系統設定是無法更改的，故您可放心使用信箱索取新的密碼；提醒您，記得將您的新密碼變更為您要的密碼，請至【會員中心】裡的【修改密碼】變更即可。

Q：我已經有黑馬會員的帳號，要如何登入會員?

  2. 點選您要登入的社群網站即可。

  2. 點選【表格下載】。
  3. 選擇您要的傳真表格，並點選下載即可。

Q：請問發生問題要如何與客服人員聯繫?

A：聯繫客服專員的步驟和說明如下：

【非會員】

  1. 請至【黑馬數位】官網，點選【客服中心】。
  2. 點選【線上回報】。
  3. 依照您的問題選擇相關類別、填寫會員資料和描述問題內容後，點選【儲存送出】即可。
  4. 提醒您，請務必填寫完整有效E-mail信箱，以免收不到客服專員的回覆。

【黑馬會員】

  1. 請至【黑馬數位】官網，點選左側【登入會員】。
  2. 輸入您的E-mail帳號、密碼，並點選【登入】。
  3. 點選【客服中心】。
  4. 點選【線上回報】。
  5. 依照您的問題選擇相關類別、填寫會員資料和描述問題內容後，點選【儲存送出】即可。

  1. E-mail服務時間：24小時。
  2. 傳真服務時間：24小時。
